# 蛇口造城记 · Shekou City Builder

基于蛇口改革开放真实历史（1979-1988）的互动叙事游戏。

玩家扮演改革开放决策者，通过四幕历史剧情体验蛇口工业区从荒芜渔村到改革开放前沿的十年历程。每一个选择都基于真实历史事件，不同决策将导向不同的历史走向和结局。

## 核心玩法

- **条件分支叙事**：Flag 驱动的事件系统，选择影响后续剧情走向
- **四维指标**：经济、政治、民心、创新——平衡四个维度决定最终结局
- **历史小游戏**：炸山填海、标语拼图、民主选举、邓小平视察路线规划
- **多结局系统**：6 种不同结局，由最终指标和历史选择共同决定
- **真实历史素材**：40+ 张历史照片、8 段视频，还原蛇口改革现场

## 四幕结构

| 幕 | 时间 | 主题 | 关键事件 |
|---|------|------|----------|
| 第一幕 | 1979 | 蛮荒开局 | 开山第一炮、4分钱改革、港商来访 |
| 第二幕 | 1980-1982 | 改革深水区 | "时间就是金钱"标语、干部聘任制、劳动合同制 |
| 第三幕 | 1983-1984 | 民主试验 | 民主选举、明华轮、邓小平视察蛇口 |
| 第四幕 | 1985-1988 | 走向成熟 | 住房商品化、招商银行、蛇口风波 |

## 技术栈

- React 18 + Vite
- GSAP (动画)
- 纯 CSS（无 UI 框架）
